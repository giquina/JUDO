# Keyboard Shortcuts Reference

Quick reference card for all keyboard shortcuts in the JUDO app.

---

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `?` | Show keyboard shortcuts overlay |
| `⌘ + K` (Mac) / `Ctrl + K` (Windows) | Open command palette |
| `Escape` | Close dialog/modal |

---

## Navigation Shortcuts

| Shortcut | Action |
|----------|--------|
| `G` then `M` | Go to Members |
| `G` then `C` | Go to Chat |
| `G` then `D` | Go to Dashboard |

---

## Action Shortcuts

| Shortcut | Action |
|----------|--------|
| `N` | Create new member/item |
| `I` | Check-in |
| `⌘ + S` / `Ctrl + S` | Save changes |
| `⌘ + Enter` / `Ctrl + Enter` | Submit form |

---

## Search Shortcuts

| Shortcut | Action |
|----------|--------|
| `/` | Focus search box |
| `⌘ + F` / `Ctrl + F` | Find on page |

---

## Inline Editing

| Shortcut | Action |
|----------|--------|
| Click or Double-click | Activate edit mode |
| `Enter` | Save changes |
| `Escape` | Cancel editing |

---

## Context Menu

| Action | Shortcut |
|--------|----------|
| Right-click | Open context menu |
| `⌘ + C` / `Ctrl + C` | Copy ID (when item selected) |
| `⌘ + E` / `Ctrl + E` | Edit item (when available) |

---

## Mobile Quick Actions

| Action | Method |
|--------|--------|
| Open FAB menu | Tap floating + button (bottom-right) |
| Close FAB menu | Tap backdrop or press Escape |

---

## Tips

1. **Discover More**: Press `?` anywhere to see all available shortcuts
2. **Search**: Use the search box in the shortcuts overlay to find specific actions
3. **Learn as You Go**: Keyboard shortcuts are shown in tooltips and menus
4. **Customize**: Shortcuts can be customized in the KeyboardShortcuts.tsx component

---

**Quick Access**: Press `?` to open the shortcuts overlay anytime!
